Q1. "We are interested in retrieving only the students' names; all the names should be in caps. ['JOHN', 'BABA', 'YAGA', 'WICK']"

Ans:-

        let studentRecords = [
        { name: "John", id: 123, marks: 98 },
        { name: "Baba", id: 101, marks: 23 },
        { name: "yaga", id: 200, marks: 45 },
        { name: "Wick", id: 115, marks: 75 },
      ];
      
      
      let UpperCase = [];
      
      UpperCase=studentRecords.map((student)=>{
        return  student.name.toUpperCase()
      })

Output:- ['JOHN', 'BABA', 'YAGA', 'WICK']

explanation:-

This code defines an array called `studentRecords` containing objects representing student records with their names, IDs, and marks. It then declares an empty array called `UpperCase`. Using the `map` function, it transforms the `studentRecords` array by extracting the names of the students and converting them to uppercase. The resulting uppercase names are stored in the `UpperCase` array.

Q4. Consider the same scenario we have discussed above, but this time we would like to know the sum total of the marks of the students..

Ans:-

        let studentRecords = [
        { name: "John", id: 123, marks: 98 },
        { name: "Baba", id: 101, marks: 23 },
        { name: "yaga", id: 200, marks: 45 },
        { name: "Wick", id: 115, marks: 75 },
      ];
      
      let sumOfMarks = studentRecords.reduce((accumulator, student) => {
        return accumulator + student.marks;
      }, 0);
      
      console.log(sumOfMarks);

Output:- 241

explanation:-

This code initializes the sumOfMarks variable to 0. The reduce method is then used to iterate over each student in the studentRecords array and accumulate their marks by adding them to the accumulator. Finally, the sum of all marks is logged to the console. In this case, the output would be 241 since the sum of 98 + 23 + 45 + 75 is 241.

Q7. This time we want to get only the names of the students who scored more than 50 marks from the same dataset used above..

Ans:-

        let studentRecords = [
        { name: "John", id: 123, marks: 98 },
        { name: "Baba", id: 101, marks: 23 },
        { name: "yaga", id: 200, marks: 45 },
        { name: "Wick", id: 115, marks: 75 },
      ];
      
      let totalMarks = studentRecords
        .map(student => {
          if (student.marks < 50) {
            return { ...student, marks: student.marks + 15 };
          } else {
            return student;
          }
        })
        .filter(student => student.marks > 50)
        .reduce((accumulator, student) => accumulator + student.marks, 0);
      
      console.log(totalMarks);

Output:- 233

explanation:-

This code first uses the map method to iterate over each student record and check if their marks are less than 50. If so, it adds a grace of 15 marks by creating a new object with the updated marks. Otherwise, it keeps the original student record unchanged. Then, the filter method is used to filter out students whose marks are still greater than 50 after the grace has been applied. Finally, the reduce method calculates the total marks by accumulating the marks of the filtered students. The resulting total marks are logged to the console
